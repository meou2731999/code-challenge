# Cascade — Currency Swap

Vite + React + TypeScript currency swap form for the 99Tech code challenge.

## Run

```bash
npm install
npm run dev
```

Open the local URL Vite prints (usually `http://localhost:5173`).

## Features

- Live token prices from [`https://interview.switcheo.com/prices.json`](https://interview.switcheo.com/prices.json)
- Token icons from [Switcheo/token-icons](https://github.com/Switcheo/token-icons)
- Searchable token picker, flip direction, USD estimates, validation
- Simulated confirm loading delay on submit
